# 📘 Mini Proje: Bilgi Kartı

## 🧾 Proje Açıklaması
Bu proje, Python'da temel veri tiplerini ve değişken kullanımını öğrenmek amacıyla oluşturulmuş basit bir "Bilgi Kartı" uygulamasıdır.

Program, kullanıcıya ait bazı bilgileri (isim, yaş, favori sayı vb.) değişkenler aracılığıyla saklar ve düzenli bir formatta ekrana yazdırır.

---

## 🚀 Özellikler
- Değişken tanımlama (string, int, float, bool)
- Formatlı çıktı (f-string kullanımı)
- Veri tiplerini görüntüleme (`type()` fonksiyonu)
- Basit koşul yapısı (`if-else`)

---

## 🧠 Kullanılan Veri Tipleri

| Veri Tipi | Açıklama |
|----------|--------|
| `str`    | Metin verisi |
| `int`    | Tam sayı |
| `float`  | Ondalıklı sayı |
| `bool`   | Doğru / Yanlış |

---

## 📌 Koddan Örnek

```python
isim = "Muharrem"
yas = 20
favori_sayi = 8
yazilimda_yeni_mi = False
```

---

## ▶️ Nasıl Çalıştırılır

1. Python yüklü olduğundan emin olun
2. Dosyayı terminalde çalıştırın:

```bash
python dosya_adi.py
```

---

## 📊 Örnek Çıktı

```
======================================
       KULLANICI BİLGİ KARTI
======================================
  İsim          : Muharrem
  Yaş           : 20
  Favori Sayı   : 8
  Yazılıma Yeni : False

```

---

## 📚 Öğrenme Notları

- Değişkenler, veriyi saklamak için kullanılır
- Python dinamik tipli bir dildir
- `print()` fonksiyonu çıktı verir
- `type()` veri tipini gösterir
