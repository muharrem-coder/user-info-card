# 📘 Mini Proje: Kullanıcı Analiz Aracı

## 🧾 Proje Açıklaması
Bu proje, Python'da string işlemleri, listeler ve temel operatörleri öğrenmek amacıyla geliştirilmiş bir kullanıcı analiz aracıdır.

Kullanıcıdan alınan bilgiler doğrultusunda basit analizler yapar ve düzenli bir rapor çıktısı oluşturur.

---

## 🚀 Özellikler
- Kullanıcıdan veri alma (`input()`)
- String işlemleri (`upper()`, `len()`)
- Liste oluşturma ve kullanma
- Koşul yapıları (`if-else`, ternary)
- Döngü kullanımı (`for`, `enumerate`)

---

## 🧠 Kullanılan Yapılar

| Yapı | Açıklama |
|------|--------|
| `str` | Metin işlemleri |
| `list` | Çoklu veri saklama |
| `int` | Sayısal veri |
| `if-else` | Karar yapıları |
| `for` | Döngü |

---

## 📌 Koddan Örnek

```python
tam_isim = isim + " " + soyisim
tam_isim_buyuk = tam_isim.upper()
isim_uzunlugu = len(isim)
```

1. Python yüklü olduğundan emin olun

## 📚 Öğrenme Notları

- String işlemleri ile metinler manipüle edilir
- Listeler birden fazla veriyi saklar
- `len()` uzunluk ölçer
- `enumerate()` listeyi index ile gezmeyi sağlar
- Koşul yapıları programın akışını kontrol eder
